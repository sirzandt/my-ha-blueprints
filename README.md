# my-ha-blueprints
## Introduction
This repo contains some blueprints I developed and use in my home assistant installation at home.

The following blueprints are available:
- Hall Light Daynight Ambient Lights. This blueprint switches lights to a configurable level or off depending on a mode, the fact whether it is dark and the fact whether it is ambient time (a schedule). The lights react to the change of the configurable level.
- Hall Light Daynight Ambient Switches. This blueprint switches switches to on / off depending on the mode, the fact whether it is dark and the fact whether it is ambient time (a schedule).
- Hall Light Motion Daynight Ambient. This blueprint switches a light to fully on (actually a configurable level), ambient (also a configurable level), night (a configurable level) or off depending on the mode, the fact whether it is dark and the fact whether to use ambient and an ambient schedule.
